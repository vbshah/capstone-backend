Correlation is defined as a linear relationship between two variables like salary and age.

 

Correlation is a float value between -1 and 1. If two variables are highly correlated, their correlation values will be close to 1 and if it's inverse, it will be close to -1. 

In our application, you can select columns for which you want to find a correlation. It will generate a heatmap and correlation matrix. Try out with an example.
